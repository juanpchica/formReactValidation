# React: Calculator

## Environment

- React Version: 16.13.1
- Node Version: 12(LTS)
- Default Port: 8000

## Project Specifications

**Read Only Files**

- src/tests/App.test.js

**Commands**

```bash

```

```bash
bash bin/env_setup && . $HOME/.nvm/nvm.sh && npm install
```

- test:

```bash
bash bin/env_setup && . $HOME/.nvm/nvm.sh && npm test
```
